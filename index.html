Privacy Policy

Effective Date: October 10, 2025

Overview

CanvasAI (“the Extension”) helps users access course materials from Canvas LMS and generate study assistance using AI models. The Extension operates only when the user chooses to activate it and processes data solely for the purpose of providing the requested functionality.

Information We Access

The Extension accesses the following data only when the user activates the extension on a Canvas page:

  • Course files, documents, and text content visible to the user on Canvas
  • User’s existing Canvas session cookies (read-only) to confirm that they are logged in

The Extension does not collect or store personal identifiers such as name, email, password, or contact information.

How Information Is Used

User content (such as files or text) may be temporarily transmitted to:

  • CanvasAI’s backend server for processing
  • Google Gemini or other AI APIs to generate summaries or study materials

This content is used only to fulfill the user’s request and is not used for any analytics, advertising, or secondary purposes.

Data Storage and Retention

  • No user data is permanently stored by the Extension.
  • Any content transmitted to CanvasAI’s backend or AI APIs is processed temporarily and automatically deleted after processing is complete.
  • No data is sold or shared with third parties.

Permissions

The Extension uses only the permissions required to function:

  • host permissions – to access Canvas pages and the CanvasAI backend
  • cookies – to read the user’s existing Canvas session for authentication
  • storage – to store temporary preferences and local settings

Third-Party Services

The Extension communicates with secure APIs (such as CanvasAI’s backend server and Google Gemini) to process user-initiated requests. These services do not store or reuse user content outside the scope of fulfilling the request.

User Control and Consent

  • Data is processed only when the user actively initiates an action.
  • Users may delete local extension data via Chrome’s extension settings.
  • Uninstalling the extension immediately stops all data processing.

Remote Code

The Extension does not fetch or execute any remote code. All executable code is packaged within the extension.
